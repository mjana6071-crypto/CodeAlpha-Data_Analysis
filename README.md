# CodeAlpha-Data_Analysis
An Data Analytics project using Python (Pandas, Seaborn) to analyze employee performance, satisfaction, and demographics. It uncovers key workforce trends and data-driven insights to understand employee attrition and help organizations optimize retention and talent management strategies.




# 📊 Employee Data Analytics using Python

## 📝 Project Overview
This project focuses on **People Analytics (HR Analytics)**, where I utilized Python to analyze employee data and uncover key insights regarding workforce demographics, performance metrics, and turnover rates. The goal is to help HR departments make data-driven decisions to improve employee retention and satisfaction.

## 🛠️ Key Features & Analysis:
* **Data Cleaning & Preprocessing:** Handling missing values and formatting categorical HR data.
* **Exploratory Data Analysis (EDA):** Identifying trends in employee performance, salary distribution, and tenure.
* **Attrition Analysis:** Investigating the main factors driving employee turnover.
* **Data Visualization:** Creating clear, insightful charts to communicate findings effectively.

## 🧰 Tech Stack:
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
